# repeat

## repeat char

creates a string with duplicating single character

```csharp
string repeat(char c, int repeat)
```

```csharp
repeat('-', 5)
// -----
```

## repeat string

creates a string with duplicating a string

```csharp
string repeat(string v, int repeat)
```

```csharp
repeat("ff", 3)
// ffffff
```
